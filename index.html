사용 안내
1. 현재 운영 중인 방이 자동으로 연결됩니다.
2. 경기 전 이름과 핸디캡만 입력하여 사전 등록하세요.
3. 라운드가 끝나면 친타수를 추가로 입력해 주세요.
4. 핸디캡은 최대 14까지만 입력 가능합니다.
5. 본인 G핸디, 친타수 다를 게 기재 시 무조건 시상에서 제외 됩니다.
